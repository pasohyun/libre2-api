# 메모 기능 운영 가이드 (노션 정리본)

## 1) 목적

온라인 가격 모니터링 업무에서 발생하는 **판단 맥락(예외/합의/이력)**을 시스템에 기록하고,
담당자 변경 시에도 동일한 기준으로 대응할 수 있도록 하기 위한 기능입니다.

- 단순 데이터 조회를 넘어, **운영 지식 축적**이 목적
- 이슈 대응 시 근거를 남겨 **재확인 비용 감소**
- 팀 단위 커뮤니케이션을 위한 **공용 메모 공간 제공**

---

## 2) 현재 구현된 메모 기능

### A. 상단 공용 메모장 (온라인 계보)

- 위치: 메인 화면 상단(로그인 후 노출)
- 용도: 팀 공지, 인수인계, 주의사항, 운영 원칙 공유
- 입력 필드
  - 요약(선택)
  - 메모 내용(필수)
- 제공 기능
  - 등록
  - 목록 조회
  - 삭제

### B. 업체 메모 취합 · 조회 카드

- 위치: 메인 대시보드 카드 영역
- 용도: 업체별 메모를 한 화면에서 빠르게 확인
- 조회 컬럼
  - 채널
  - 판매처
  - 요약
  - 작성일
  - 본문 미리보기
- 제공 기능
  - 새로고침
  - 세부(해당 판매처 상세로 이동)
  - 삭제
  - **빠른 등록(신규 추가)**
    - 판매처 선택
    - 요약(선택)
    - 메모 내용(필수)
    - 빠른 등록 버튼

### C. 판매처 상세의 업체 메모

- 위치: 판매처 상세 페이지 내 `판매처 메모` 카드
- 용도: 특정 업체 단위의 상세 기록
- 입력 필드
  - 요약(선택)
  - 메모 내용(필수)
- 제공 기능
  - 등록
  - 목록 조회
  - 삭제

## 3) 현업 사용 방법 (실무 기준)

### 3-1. 업체 메모를 빠르게 등록하는 방법 (권장)

1. 메인 화면에서 `업체 메모 취합 · 조회` 카드 확인
2. 카드 상단 `빠른 등록` 영역에서 판매처 선택
3. 요약/메모 내용 작성
4. `빠른 등록` 클릭
5. 등록 후 목록 갱신 및 해당 판매처 상세로 이동해 추가 확인

**언제 쓰면 좋은가**
- 미준수 이슈 발견 직후
- 전화/메일 협의 결과를 바로 남길 때
- 동일 이슈 재발 가능성이 높을 때

### 3-2. 판매처 상세에서 등록하는 방법

1. 메인 또는 취합 카드에서 판매처 `세부` 이동
2. `판매처 메모` 카드에 요약/내용 입력
3. `이 판매처에 메모 등록` 클릭

**언제 쓰면 좋은가**
- 특정 판매처 이력만 집중해서 정리할 때
- 타임라인/가격 변동 데이터를 보면서 함께 기록할 때

### 3-3. 공용 메모장 사용 방법

1. 메인 상단 `전체 운영 메모 (온라인 계보)` 확인
2. 공지/원칙/인수인계 내용 입력
3. 필요 시 과거 공용 메모를 조회하거나 삭제

**권장 사용 예시**
- "이번 주 모니터링 기준 변경"
- "특정 거래처 예외 처리 기준"
- "담당자 교체 시 반드시 확인할 항목"

## 4) 작성 규칙(권장)

메모 품질을 위해 아래 템플릿 사용 권장:

- 요약: `[거래처명] [핵심 이슈] [상태]`
  - 예: `[닥다몰] 기준가 미준수 반복 [협의중]`
- 본문 4요소
  1. 상황: 무엇이 발생했는지
  2. 근거: 기준가/시점/링크/증빙 여부
  3. 조치: 누구에게 무엇을 했는지
  4. 다음 액션: 재확인 일정/담당자

---

## 5) 권한/접속 조건

- 대시보드 로그인(JWT 인증) 후 사용 가능
- 메모 API는 인증 사용자만 접근 가능
- 현재 운영 API 기준 메모 엔드포인트 제공 중
  - `/memos/global`
  - `/memos/vendor`
  - `/memos/vendors/aggregate`
  - `DELETE /memos/{memo_id}`

## 6) 자주 묻는 질문 (FAQ)

### Q1. 취합 카드에서 바로 등록이 안 될 때

- 판매처를 선택했는지 확인
- 메모 내용(본문) 입력했는지 확인
- 로그인 세션 만료 여부 확인(재로그인)

### Q2. 저장은 됐는데 바로 안 보일 때

- 카드의 `새로고침` 버튼 클릭
- 브라우저 강력 새로고침 (`Ctrl + F5`)

### Q3. 메모를 잘못 쓴 경우

- 카드의 `삭제` 버튼으로 제거 후 재등록

- 보고/커뮤니케이션 품질 표준화
- 인수인계 리스크 감소
